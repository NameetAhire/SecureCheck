API Documentation

Our API allows you to check if email addresses have been compromised in known data breaches for single, real-time lookups.

Endpoint

POST /api/check-email

Request Body

Content-Type: application/json

{
    "email": "user@example.com"
}

Response (200 OK)

{
    "email": "user@example.com",
    "status": "breached",
    "breaches": [
        "LinkedIn (2012)",
        "Adobe (2013)"
    ]
}
